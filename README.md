# Academia de Magia – Jogo RPG em Swift (Terminal)

Este é um projeto simples de RPG feito em Swift, onde simula-se uma Academia de Magia em que o Aprendiz tem a oportunidade
de aprender novos feiticos e testar seus conhecimentos num Simulacro aleatorio. O jogo utiliza recursos
básicos como arrays, sets, dicionários, tuplas, funções, loops e tratamento de erros.

Como jogar

Ao iniciar o jogo, digite seu nickname no terminal. Depois, o menu principal será
exibido, digite o numero da opção para acessar o menu:

1.  Aprender Feitico
2.  Ver Grimorio
3.  Enfrentar um Simulacro
4.  Sair do jogo

Aprender Feitiço

Mostra todos os feitiços disponíveis e permite aprender um deles.

Ver Grimório

Exibe os feitiços aprendidos.

Enfrentar um Simulacro

Inicia uma batalha contra um inimigo simulacro aleatório. 
Você escolhe um feitiço, gasta mana e causa dano.
O inimigo contra-ataca até alguém ser derrotado.
Dica: Nao esqueca de aprender um feitico antes de lutar. Mais de um feitico pode lhe ajudar em batalhas mais dificeis

Sair do jogo

Encerra o programa.
